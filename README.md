# insa-route
## a routing tool using the insa interface

### Requirements
- Ruby >=2.3
- Rubygem "bundler" (`gem install bundler`)

### Installation in folder via Github:
```
$ git clone https://github.com/izumariu/insa-route.git
$ bundle install
$ chmod u+x insa-route
```
